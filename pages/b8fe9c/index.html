# 快捷键

command + , 可以打开编译器设置

⌘（command）、⌥（option/alt）、⇧（shift）、⌃（control）、⎋（esc）

快捷键 作用
command + option + L 格式化
command + c / command + v 复制/粘贴
command + z / command + shift + z 撤销/前进
command + delete 删除行
command + d 重复行
control + shift + r 运行当前py文件
command + r 替换
command + 4 打开运行窗口
command + 5 打开调试窗口
command + b / command + 点击 都可跳转到函数定义的位置
command + p 显示函数的参数信息 (不消失的那种)
自动format补全 {变量名
连续按两次shift 全局搜索框
option + Enter 选中黄色警告的该行使用
command + 1 打开关闭左侧的项目导航
option + ⬆️ ⬇️ 扩大缩小代码选区
option + shift + ⬆️ ⬇️ 代码行上下移动
command + shift + ⬆️ ⬇️ 上下交换代码块的位置 (光标在def处)
command + option + ⬅️ ➡️ 跳转到上一回\下一回操作的地方
command + option + ➕ ➖ 光标放到class那行的末尾, 快捷键展开收起class里的所有函数
(切换到英文输入法的状态下)
command + shift + ➕ ➖ 光标放到任意位置, 快捷键将模块中所有class def 展开收起
(大写锁定)

Ps: 光标在该行的任意位置都可.
